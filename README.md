This is a [Next.js](https://nextjs.org) project with an App Router demo of a reusable **DataGrid** (shadcn UI, AR/EN via next-intl).

## Getting Started

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) (redirects to `/en`). Locale routes: `/en`, `/ar`.

## Docs

- [DataGrid usage](src/components/data-grid/data-grid.md) — columns, selection, actions, expandables, client/server modes, i18n
- [`useFetch` hook](src/hooks/use-fetch.md) — client-side fetch with loading/error/abort
